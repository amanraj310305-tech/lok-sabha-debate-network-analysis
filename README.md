# Lok Sabha Debate Network Analysis

This project performs network-based text analysis on Lok Sabha debate transcripts.

## Features
- Text preprocessing using NLTK
- Word co-occurrence network creation
- Community detection using Louvain method
- Polarization measurement via modularity
- Bridging term identification using betweenness centrality
- Visualization with NetworkX and Matplotlib

## Technologies
- Python
- Pandas
- NetworkX
- NLTK
- Matplotlib
- python-louvain

## How to Run

Install dependencies:

pip install -r requirements.txt

Run:

python analysis.py

## Output
- Network visualization
- Community size chart
- CSV with word metrics
